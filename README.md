# Лечение зубов в Китае — фронт + бэкенд

Сайт-лендинг для услуги сопровождения и организации поездок на стоматологическое лечение в Китай.

- **Фронтенд** — React + TypeScript + Tailwind CSS; моки в `src/data.ts`, форма может ходить в API.
- **Бэкенд** — FastAPI в папке **`backend/`** (JWT, заявки, клиника, процедуры). В репозитории на GitHub папка **backend** видна в корне — откройте её на сайте или клонируйте репозиторий целиком.

## Backend (виден в общем репозитории)

| Путь | Назначение |
|------|------------|
| **`backend/`** | Полный код API: `app/main.py`, роутеры, `requirements.txt` |
| **`backend/README.md`** | Как установить зависимости и запустить uvicorn (порт 8002, Swagger `/docs`) |
| **`backend/.env.example`** | Скопировать в `.env` — JWT, админ, CORS |

Запуск локально:
```bash
cd backend
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt && cp .env.example .env
uvicorn app.main:app --host 0.0.0.0 --port 8002 --reload
```

## Структура фронтенда (основное)

| Файл | Назначение |
|------|------------|
| `index.html` | Единая точка входа HTML |
| `src/main.tsx` | Подключение React и рендер в `#root` |
| `src/App.tsx` | Роутер (React Router): все маршруты и Layout |
| `src/index.css` | Стили Tailwind (base, components, utilities) |
| `src/data.ts` | Моковые данные: навигация, тексты, пакеты, отзывы, FAQ, контакты, юридика |
| `src/Layout.tsx` | Общий макет: шапка (навигация), контент, подвал со ссылками и контактами |
| `src/Pages.tsx` | Все страницы: Главная, Услуги, Как это работает, Цены, Партнёры, Отзывы, FAQ, Блог, Контакты (с формой), юр. страницы, 404 |

Конфигурация: `package.json`, `vite.config.ts`, `tailwind.config.js`, `postcss.config.js`, `tsconfig.json`.

## Страницы и маршруты

- `/` — Главная (hero, преимущества, процесс, пакеты)
- `/uslugi` — Услуги (транспорт, сопровождение, подбор клиники, программы)
- `/process` — Как это работает (6 шагов)
- `/prices` — Цены и пакеты (Базовый, Стандарт, Премиум)
- `/partnery` — Партнёры / клиники (карточки)
- `/otzyvy` — Отзывы
- `/faq` — Вопросы и ответы (раскрывающиеся блоки)
- `/blog` — Список статей блога
- `/blog/:id` — Одна статья (мок)
- `/kontakty` — Контакты + форма заявки (имя, телефон, email, город, дата, цель, файлы, комментарий, согласие)
- `/privacy` — Политика конфиденциальности
- `/offer` — Договор-оферта
- `/medical-disclaimer` — Медицинский отказ
- любая другая — 404 с ссылками на Главная / Услуги / Контакты

## Запуск

1. Установить зависимости:
   ```bash
   npm install
   ```

2. Запустить dev-сервер:
   ```bash
   npm run dev
   ```
   Откроется локальный адрес (обычно `http://localhost:5173`). Навигация по сайту — через шапку.

3. Сборка для продакшена:
   ```bash
   npm run build
   ```
   Результат в папке `dist/`.

4. Просмотр собранной версии:
   ```bash
   npm run preview
   ```

## Форма заявки

Поля по ТЗ: Имя, Фамилия, Телефон (обязательно), Email, город вылета, дата поездки, цель, загрузка файлов, комментарий, согласие. Бэкенд принимает заявки через API (когда фронт настроен на `API_BASE`). Локально без API — показ сообщения об успехе.

## Дальнейшие шаги

- Настроить `API_BASE` на URL бэкенда и CORS в `backend/.env` (`FRONTEND_ORIGIN`).
- При необходимости вынести страницы в отдельные файлы под `src/pages/`.
